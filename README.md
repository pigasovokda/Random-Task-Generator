# Random Task Generator

**Автор:** Пигасов Дмитрий

## Описание

Random Task Generator — это GUI-приложение на Python для генерации случайных задач из заданного списка. Приложение помогает разнообразить повседневные дела, предлагая случайные задачи для учёбы, спорта или работы.

### Основные возможности

- Генерация случайной задачи из предопределённого списка
- Фильтрация задач по типу (учёба, спорт, работа, другое)
- Добавление новых задач с указанием типа
- Сохранение истории сгенерированных задач с указанием времени
- Сохранение всех данных (задачи + история) в JSON-файл
- Обработка ошибок (проверка на пустую строку при добавлении задач)

## Технологии

- Python 3.7+
- tkinter (встроенный GUI-фреймворк)
- random (генерация случайных чисел/задач)
- json (сохранение и загрузка данных)

### Требования
